### :dart: 特点

- **mockjs**生成无序的随机数据，专注于模板本身的效果。
- 模板支持**语言切换**，减少因为语言原因导致的UI问题。
- **响应式**，默认兼容`mac`、`ipad`、`iphone`。
- **daisyui**，默认支持29中主题，也可以[自定义主题](https://daisyui.com/theme-generator/)，无需修改模板代码。

## :running: 本地运行

克隆项目

```bash
  git clone https://github.com/meetqy/css.ist.git
```

进入项目目录

```bash
  cd css.ist
```

安装依赖

```bash
  yarn install
```

启动服务

```bash
  yarn dev
```
